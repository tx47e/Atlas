# Harta template-uri Vault

| Template | Folosire |
| --- | --- |
| `Template_Notite_Concept.md` | Concept numerologic de baza: definitie, cuprins, dependente si utilizare in lucrare. |
| `Template_Notite_Calcule.md` | Formula, pasi de calcul, observatii si limite ale metodei. |
| `Template_Notite_Exemple_Calcul.md` | Unul sau mai multe exemple complete de calcul validate. |
| `Template_Notite_Arhetip.md` | Descrierea unei vibratii sau a unui arhetip: lumina, umbra, lectii si directii. |
| `Template_Notite_Subcapitol.md` | Nota de sprijin pentru un subcapitol sau o componenta a unui concept mai mare. |
| `Template_Notite_Sinteza.md` | Sinteza transversala: scop, date, indicatori, corelari si concluzie. |
| `Template_Lucrare_Numerologica.md` | Template de lucrare numerologica generala; nu se foloseste pentru note de Vault. |
| `Template_Lucrare_Numerologica_Examen.md` | Template de lucrare de examen; nu se foloseste pentru note de Vault. |
| `Plan_Template_Lucrare_Numerologica_Narativa.md` | Plan de structurare narativa pentru Agent Lucrari; nu se foloseste pentru note de Vault. |
| `Cuprins_Lucrare_Numerologica.md` | Cuprins de referinta pentru lucrari; nu se foloseste pentru note de Vault. |

## Fallback

Cauta mai intai in `vault/Numerologie/` dupa titlu, taguri, `tip:` si titluri de sectiune. Alege conceptul cu acelasi scop functional, nu doar cu aceleasi cuvinte. Preia numai structura, nu valorile sau interpretarile persoanei din acel precedent.
