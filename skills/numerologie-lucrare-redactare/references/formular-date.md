# Formular de intrare

Skill părinte: [[skills/numerologie-lucrare-redactare/SKILL|numerologie-lucrare-redactare]]

Colecteaza urmatoarele date. Campurile marcate optional se omit cand nu se aplica.

```text
PERSOANA ANALIZATA
Nume si prenume:
Prenume activ:
Data nasterii: ZZ.LL.AAAA
Nume anterior: [optional]
Gen: M / F

LUCRARE
Template: examen / scurt / alt tip disponibil
Exprimare: conversational / formal
Nivel de detaliere: scurt / mediu / amplu
Intrebare principala: cariera / iubire / bani / faima / alta intrebare
Interval de ani: complet (0-108 ani) / specific: [an-varsta de inceput] - [an-varsta de final]

RELATIE [optional]
Nume complet:
Prenume activ:
Data nasterii: ZZ.LL.AAAA
Gen: M / F
Tipul relatiei:
```

Inainte de calcul, confirma orice data ambigua. Nu deduce genul, prenumele activ sau tipul relatiei doar din nume.
