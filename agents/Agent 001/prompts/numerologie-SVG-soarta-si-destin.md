# Prompt - numerologie-SVG-soarta-si-destin

Foloseste skill-ul `numerologie-SVG-soarta-si-destin` pentru a crea sau adapta graficul Soarta si Destin pentru persoana indicata.

Citeste complet `SKILL.md`, foloseste `assets/reference.svg` pentru compozitie si verifica valorile in `vault/Numerologie/`. Pastreaza legenda, zona de confort si logica vizuala a graficului conform referintei.

Valorile Soarta, Destin si zona de confort trebuie sa fie consecvente cu orice harta suprapusa generata pentru aceeasi persoana. Pentru calcule cu traseu lung, scrie pasii cu `->`.

La final valideaza SVG-ul ca XML si verifica vizual punctele, liniile, etichetele si legenda.
