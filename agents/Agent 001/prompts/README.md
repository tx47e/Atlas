# Prompturi Agent 001

Acest director contine prompturile reutilizabile pentru skill-urile SVG gestionate de Agent 001.

Regula generala: fiecare prompt incepe cu formularea `Esti Agent 001`. Inainte de generare, Agent 001 citeste complet skill-ul asociat si urmeaza workflow-ul operational curent. Pentru generarea uzuala, ruleaza exclusiv scriptul indicat de skill, fara recalcul manual, fara consultarea `vault/Numerologie/` si fara modificarea manuala a SVG-ului rezultat. Fiecare SVG final trebuie sa contina watermark-ul exact `Atlas Numerologie`. Referinta SVG si documentatia numerologica se verifica separat numai cand utilizatorul cere mentenanta sau sincronizarea metodei.
