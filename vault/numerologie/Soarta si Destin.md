## Descriere

Soarta si Destin compara doua linii ale vietii:

- soarta, adica linia de conditionare, cadrul primit si drumul initial;
- destinul, adica directia de implinire si varianta de urcat catre potential.

Cele doua rezultate nu se reduc la o singura cifra. Ele se pastreaza ca numere grafice de 7 cifre si se citesc impreuna prin zona de confort, puncte de intalnire, puncte de rascruce si ritmuri de 10 sau 12 ani.

Interpretarea comparativa dintre cele doua linii poate fi numita si tema vietii, atunci cand accentul cade pe concluzia de sinteza.

---
## Formula de calcul

### Calculul sortii

Soarta se calculeaza prin formula:

```text
soarta = ZZLL x AAAA
```

Unde:

- `ZZ` este ziua nasterii scrisa cu doua cifre;
- `LL` este luna nasterii scrisa cu doua cifre;
- `AAAA` este anul nasterii scris cu patru cifre.

Pasi:

1. Se scrie ziua cu doua cifre.
2. Se scrie luna cu doua cifre.
3. Se unesc ziua si luna in blocul `ZZLL`.
4. Se inmulteste `ZZLL` cu anul nasterii `AAAA`.
5. Daca rezultatul are mai putin de 7 cifre, se completeaza cu zerouri in fata.
6. Numarul de 7 cifre se foloseste pentru graficul sortii.

### Calculul destinului grafic

Destinul se calculeaza din aceeasi structura a datei, dar cu zerourile inlocuite cu 1:

```text
destin = ZZLL_ajustat x AAAA_ajustat
```

Pasi:

1. Se scrie ziua cu doua cifre.
2. Se scrie luna cu doua cifre.
3. Se unesc ziua si luna in blocul `ZZLL`.
4. Se scrie anul nasterii in forma `AAAA`.
5. Se verifica daca `ZZLL` si `AAAA` contin cifra 0.
6. Daca nu exista niciun 0, destinul grafic este egal cu soarta si nu se mai recalculeaza separat.
7. Daca exista 0, in `ZZLL` si `AAAA`, fiecare 0 se inlocuieste cu 1.
8. Se inmulteste blocul `ZZLL` ajustat cu anul ajustat.
9. Daca rezultatul are mai putin de 7 cifre, se completeaza cu zerouri in fata.
10. Numarul de 7 cifre se foloseste pentru graficul destinului.

### Zona de confort

Zona de confort se calculeaza separat pentru soarta si pentru destin:

```text
zona_de_confort = suma_cifrelor(numar_grafic) / 7
```

Interpretare:

- sub zona de confort apare pasivitate, presiune sau lipsa de chef;
- in zona de confort persoana functioneaza firesc;
- peste zona de confort apare efortul de crestere si dinamica evolutiva.

---
## Interpretare generala

Soarta si Destin se citeste prin comparatie:

- unde liniile sunt apropiate;
- unde se indeparteaza;
- unde omul este in zona de confort;
- unde apar puncte de intalnire;
- unde apar puncte de rascruce;
- ce cere cadrul primit;
- ce cere directia de implinire.

Daca liniile sunt apropiate, persoana poate simti mai usor legatura dintre ce primeste si ce are de construit. Daca liniile sunt departate, apare maturizarea prin alegere: persoana trebuie sa uneasca doua ritmuri diferite.

Daca soarta este mai confortabila decat destinul, tema poate fi depasirea inertiei si iesirea din familiar. Daca destinul este mai confortabil decat soarta, tema poate fi recuperarea increderii si folosirea potentialului care se deschide mai usor decat cadrul initial.

## Exemplu de calcul

### Exemplu 1

Date initiale:

- Data nasterii: 09.02.1969.

Calcul soarta:

```text
09.02.1969 -> ZZLL = 0902
0902 x 1969 = 1776038
```

Zona de confort soarta:

```text
1 + 7 + 7 + 6 + 0 + 3 + 8 = 32
32 / 7 = 4,57
```

Calcul destin:

```text
0902 -> 1912
1969 -> 1969
1912 x 1969 = 3764728
```

Zona de confort destin:

```text
3 + 7 + 6 + 4 + 7 + 2 + 8 = 37
37 / 7 = 5,28
```

Interpretare scurta:

Soarta are zona de confort 4,57, iar destinul urca la 5,28. Tema vietii poate fi formulata ca trecere de la un cadru primit mai stabil si prudent catre o directie de implinire care cere mai multa miscare, asumare si crestere.

### Exemplu 2

Date initiale:

- Data nasterii: 01.03.1985.

Calcul soarta:

```text
0103 x 1985 = 204455
0204455
```

Rezultatul are 6 cifre, deci se completeaza cu zero in fata.

Calcul destin:

```text
0103 -> 1113
1985 -> 1985
1113 x 1985 = 2209305
```

Rezultatul destinului are deja 7 cifre.

---
## Observatii

- Destinul din aceasta metoda nu este acelasi lucru cu [[Vibratia Destinului]].
- Destinul grafic nu se calculeaza din nume si nu se confunda cu [[Numarul de Exprimare]].
- Soarta si destinul se pastreaza ca numere grafice de 7 cifre.
- Ciclul de 10 ani se foloseste mai ales pentru predominanta masculina, iar ciclul de 12 ani pentru predominanta feminina.

## Utilizare in lucrare

Se foloseste pentru capitolul de tema a vietii, comparand cadrul primit cu directia constienta de implinire.
