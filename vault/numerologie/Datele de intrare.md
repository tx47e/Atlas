## Descriere

Se va preluat toate datele de la persoana: 
- data nasterii
- nume si prenume
- nume activ
- intrebari

## Data nasterii

- Data nasterii este in format: ***ZZ.LL.AAAA***

## Nume si prenume

- Formatul numelui este: ***Nume, Prenume***
- Persoana poate avea Nume compus. ex: ***Nume-Nume***
- Persoana poate avea mai multe prenume. ex: ***Prenume-Prenume***

## Nume activ

- Numele activ este acel nume pe care persoana il foloseste cel mai des. Cum este strigat/a cel mai des.
## Intrebari

- clientul poate avea intrebari generale sau mai specifice
