## Descriere

[[Matricea Datei de Nastere|Matricea datei de nastere]] este forma folosita in proiect pentru [[Matricea Datei de Nastere|Patratul lui Pitagora]]: o matrice 3x3 construita din cifrele datei de nastere si patru numere de lucru. Sirul complet de cifre folosit pentru matrice este [[Cod Numerologic Personal|codul numerologic personal]].

Ea arata distributia energiei native pe noua pozitii: caracter, energie, expresie, corp, centru, munca, spiritualitate, social si intelect.

---
## Formula de calcul

Data se scrie in forma:

```text
ZZLLAAAA
```

Se folosesc cifrele brute ale datei, fara 0, plus patru numere de lucru:

```text
N1 = suma tuturor cifrelor datei
N2 = reducerea lui N1
N3 = N1 - 2 x prima cifra nenula din ziua nasterii
N4 = reducerea lui N3
```

Cifrele din [[Cod Numerologic Personal|codul numerologic personal]]:

```text
data nasterii + N1 + N2 + N3 + N4
```

se aseaza in matrice, fara cifra 0.

### Schema matricei

```text
1 | 4 | 7
2 | 5 | 8
3 | 6 | 9
```

## Interpretare generala

| Pozitie | Tema |
| --- | --- |
| [[Vibratia 1|1]] | caracter, vointa, identitate |
| [[Vibratia 2|2]] | energie emotionala si vitala |
| [[Vibratia 3|3]] | expresie, comunicare, talent |
| [[Vibratia 4|4]] | corp, disciplina, baza practica |
| [[Vibratia 5|5]] | centru, echilibru, intuitie practica |
| [[Vibratia 6|6]] | munca, responsabilitate, familie |
| [[Vibratia 7|7]] | spiritualitate, noroc, protectie |
| [[Vibratia 8|8]] | social, dreptate, organizare |
| [[Vibratia 9|9]] | intelect, memorie, intelepciune |

Frecventa conteaza: cifra absenta indica tema de dezvoltare, o aparitie indica potential disponibil, doua aparitii intensifica domeniul, iar trei sau mai multe aparitii arata accent puternic sau supraincarcare.

## Exemplu de calcul

Data:

```text
24.04.1982 -> 24041982
```

Numere de lucru:

```text
N1 = 2 + 4 + 0 + 4 + 1 + 9 + 8 + 2 = 30
N2 = 3 + 0 = 3
N3 = 30 - (2 x 2) = 26
N4 = 2 + 6 = 8
```

Sir complet:

```text
24041982 + 30 + 3 + 26 + 8
```

Cifre introduse:

```text
2, 4, 4, 1, 9, 8, 2, 3, 3, 2, 6, 8
```

Matrice:

```text
1   | 44 |
222 |    | 88
33  | 6  | 9
```

## Observatii

- Zero nu se introduce in matrice.
- Repetitia conteaza.
- Absenta unei cifre nu este defect, ci tema de dezvoltare.
- Matricea este instrument simbolic, nu diagnostic psihologic.
