# Tendinta

Tendinta se stabileste exclusiv dintr-un vector diagonal plin al matricei: `159` sau `357`.

- un vector diagonal este plin cand toate cele trei casute contin cel putin o cifra;
- dintre diagonalele pline, tendinta este cea cu cea mai mare cantitate totala de cifre;
- daca au aceeasi cantitate, se citesc ca tendinte asociate;
- daca nu exista nicio diagonala plina, rubrica Tendinta se omite din lucrare.

Tendinta nu se alege dupa valoarea numerica a vectorului si nu se confunda cu fixația: fixația foloseste numai vectorii orizontali plini `147`, `258`, `369`.
