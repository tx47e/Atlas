## Descriere

[[Punti|Puntile]] arata diferenta dintre doua vibratii si energia prin care se poate face trecerea de la o zona la alta.

Ele nu inlocuiesc vibratiile comparate, ci arata energia de legatura dintre ele. O punte poate explica ce ajuta persoana sa faca pace intre interior si exterior, intre expresie si destin sau intre planul personal si planul cosmic.

---
## Ce indica

- Distanta dintre doua vibratii.
- Energia de tranzitie dintre doua zone ale profilului.
- Modul prin care se poate armoniza o tensiune.
- Diferenta dintre interior si exterior.
- Diferenta dintre vibratiile personale si [[Vibratia Destinului]].

---
## Rol in analiza

[[Punti|Puntile]] se folosesc pentru a interpreta relatiile dintre calcule.

In analiza, ajuta la:

- intelegerea diferentei dintre [[Vibratie Interioara]] si [[Vibratie Exterioara]];
- calcularea trecerii catre [[Vibratia Destinului]];
- identificarea unei energii de integrare;
- formularea recomandarilor din [[Concluzii]].

## Formula de calcul

Puntea este diferenta dintre doua vibratii. Se foloseste valoarea absoluta.

```text
punte = valoare_absoluta(vibratie_a - vibratie_b)
```

### Puntea interior - exterior

```text
punte interior-exterior =
  valoare_absoluta(vibratia_interioara - vibratia_exterioara)
```

Aceasta arata ce ajuta persoana sa faca pace intre cine este in interior si rolul social pe care il are de jucat.

### Punti catre destin

```text
punte interior-destin = |vibratia_interioara - vibratia_destinului|
punte exterior-destin = |vibratia_exterioara - vibratia_destinului|
punte cosmic-destin = |vibratia_cosmica_totala - vibratia_destinului|
```

### Rezultat final

| Punte | Calcul | Rezultat | Sens |
| --- | --- | --- | --- |
| Interior - exterior |  |  |  |
| Interior - destin |  |  |  |
| Exterior - destin |  |  |  |
| Cosmic - destin |  |  |  |

## Interpretare generala

Rezultatul puntii se citeste prin vibratia corespunzatoare:

- [[Vibratia 0|0]]: nu exista distanta de traversat; cele doua vibratii se sustin direct.
- [[Vibratia 1|1]]: trecerea se face prin initiativa, decizie si afirmare.
- [[Vibratia 2|2]]: trecerea se face prin cooperare, rabdare si sensibilitate.
- [[Vibratia 3|3]]: trecerea se face prin exprimare, relationare si creativitate.
- [[Vibratia 4|4]]: trecerea se face prin structura, disciplina si constructie.
- [[Vibratia 5|5]]: trecerea se face prin flexibilitate, schimbare si experienta.
- [[Vibratia 6|6]]: trecerea se face prin grija, armonizare si responsabilitate.
- [[Vibratia 7|7]]: trecerea se face prin analiza, introspectie si discernamant.
- [[Vibratia 8|8]]: trecerea se face prin forta, organizare si asumarea resurselor.
- [[Vibratia 9|9]]: trecerea se face prin compasiune, sinteza si eliberare.

## Exemplu de calcul

### Exemplu 1

Date initiale:

- [[Vibratie Interioara|Vibratia interioara]]: [[Vibratia 2|2]].
- [[Vibratie Exterioara|Vibratia exterioara]]: [[Vibratia 8|8]].

Calcul:

```text
|2 - 8| = 6
```

Interpretare scurta:

Prin energia lui [[Vibratia 6|6]], de exemplu prin armonizare, grija si responsabilitate, persoana poate impaca interiorul sensibil de [[Vibratia 2|2]] cu rolul exterior puternic de [[Vibratia 8|8]].

### Exemplu 2

Date initiale:

- [[Vibratie Interioara|Vibratia interioara]]: [[Vibratia 6|6]].
- [[Vibratie Exterioara|Vibratia exterioara]]: [[Vibratia 6|6]].

Calcul:

```text
|6 - 6| = 0
```

Interpretare scurta:

Rezultatul [[Vibratia 0|0]] arata ca nu exista o punte de trecut. Interiorul si exteriorul se sustin direct.

## Observatii

- Rezultatul 0 arata ca cele doua vibratii comparate sunt identice.
- Puntea nu inlocuieste vibratiile comparate.
- Puntea arata energia care poate ajuta persoana sa puna doua zone in relatie.
- Se foloseste valoarea absoluta, deci ordinea vibratiilor nu schimba rezultatul.
