# Vibratia destinului

Vibratia destinului se calculeaza din ziua, luna si anul nasterii adunate ca
numere. Ea descrie directia majora derivata din data de nastere.

Acest calcul este diferit de `destinul` din capitolul grafic `Soarta si destin`,
unde se foloseste formula `ZZLL ajustat x AAAA ajustat`.

## Formula

```text
vibratia destinului = reducere_numerologica(ziua nasterii + luna nasterii + anul nasterii)
```

## Rezultat

- o vibratie intre 1 si 9;
- traseul complet de reducere, de exemplu `24 + 4 + 1982 -> 2010 -> 3`;
- interpretare pe cifra finala si nuantare prin suma initiala, pragurile de
  reducere si eventualele valori intermediare.
