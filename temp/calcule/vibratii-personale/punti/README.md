# Punti

Puntile arata diferenta dintre doua vibratii si energia prin care se poate face
trecerea de la o zona la alta.

Interpretarea puntii se face din fisierele complete ale vibratiilor implicate:
vibratia de pornire, vibratia de destinatie si vibratia rezultatului puntii. Nu
se foloseste o lista scurta de sensuri ca baza principala.

In lucrare, capitolul trebuie sa includa tabelul rezultat final completat si o
interpretare scrisa pentru fiecare punte.
