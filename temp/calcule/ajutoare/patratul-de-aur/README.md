# Patratul de aur 3x3

Patratul de aur 3x3 este o metoda din capitolul `Ajutoare`. El foloseste
asezarea casutelor din patratul lui Pitagora si distribuie noua numere
consecutive, pornind de la ziua nasterii.

Fiecare rand, coloana si diagonala trebuie sa dea aceeasi suma. Aceasta suma
este egala cu valoarea din centru inmultita cu `3`.
