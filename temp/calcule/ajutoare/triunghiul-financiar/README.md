# Triunghiul financiar

Triunghiul financiar face parte din capitolul `Ajutoare`. El este reprezentat
ca un triunghi isoscel rosu, cu varful in sus, construit din patru cifre
derivate din data nasterii.

## Fisiere

- `01-metoda.md`: regula de calcul si reprezentare.
- `02-exemple.md`: exemplu complet pentru data `27.11.1973`.
